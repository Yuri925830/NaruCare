# NaruCare Agent UI

本套设计同时包含电脑端与手机端，共 52 张独立 PNG 图片；每个端各 26 张，每张图片只展示一个页面，没有多页面拼图。

## 画布规格

- 电脑端：1440 × 900 px
- 手机端：430 × 932 px

## 页面顺序

1. 登录
2. 注册
3. Naru 首次欢迎
4. 未建立就诊卡拦截提示
5. 建立就诊卡
6. 建卡成功后的 Agent 对话
7. 附近医院
8. 就诊流程
9. 导航路线
10. 翻译对话
11. 真人陪诊师
12. 危急情况确认
13. 119 呼叫与韩语循环播报
14. 我的
15. 真人陪诊须知
16. 陪诊师条件筛选
17. 陪诊师详情
18. 陪诊沟通
19. 等待陪诊师确认
20. 支付定金
21. 陪诊师已到达
22. 陪诊服务进行中
23. 服务结束与评价
24. 就诊记录
25. 未建立就诊卡直接呼叫 119
26. 语言选择

## 核心设计规则

- 视觉以 Naru 的奶油白、暖沙色、可可棕和少量蜜桃色为主。
- 卡片、按钮和状态组件使用珍珠高光、软质渐变、边缘反光与悬浮阴影，保持与 3D Naru 一致的立体材质。
- 电脑端采用左侧固定导航栏，手机端采用五项底部导航。
- 未建立就诊卡时，除紧急呼叫外的服务显示为受限或锁定。
- 紧急呼叫采用独立的珊瑚红与深海军蓝，提高识别速度和紧迫感。
- 所有角色胸牌名称统一为 Naru。
